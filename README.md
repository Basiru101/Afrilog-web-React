# Afrilog Web React

A modern logistics and cargo-tracking web application for Cameroon and Central African regional supply chains.

## 🚀 Features
- Cargo tracking system
- Delivery booking modal
- Responsive design
- React + Vite setup

## 🛠️ Technologies
- React
- Vite
- CSS3
- JavaScript
